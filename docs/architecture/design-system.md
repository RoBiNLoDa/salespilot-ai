# SalesPilot AI - Design System

## Objetivo

Definir las reglas visuales de SalesPilot AI para garantizar una experiencia consistente, escalable y mantenible.

---

# Filosofía

SalesPilot AI será una aplicación SaaS moderna enfocada en productividad.

Principios:

- Simplicidad
- Consistencia
- Accesibilidad
- Legibilidad
- Diseño limpio
- Preparado para modo claro y oscuro

# Componentes

Todos los componentes deberán soportar:

- Light Theme
- Dark Theme
- Responsive Design
- Accesibilidad

# Principios

Nunca usar colores hardcodeados.

Siempre utilizar variables del tema.

No duplicar estilos.

Los componentes deberán ser reutilizables.
